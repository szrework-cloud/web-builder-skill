# Page Structures Reference

## Standard Multi-Page Website Structure

### Homepage (index.html)
- Hero section with headline + CTA
- Services/offerings overview (3-6 cards)
- Social proof (testimonials, client logos, stats)
- About teaser with link to full page
- Recent work/portfolio preview
- Contact CTA section
- Footer with links, contact info, social media

### About Page (about.html)
- Company story / mission
- Team section with photos and roles
- Values / approach
- Timeline / milestones (optional)
- Certifications / awards (if applicable)

### Services Page (services.html)
- Service categories with descriptions
- Pricing tiers (if applicable)
- Process / how it works section
- FAQ accordion
- CTA to contact

### Portfolio / Realisations (portfolio.html)
- Filterable project grid
- Project cards with image, title, category
- Individual project detail views (optional)

### Contact Page (contact.html)
- Contact form (name, email, phone, message, subject)
- Google Maps embed or address
- Business hours
- Phone, email, social links
- Optional: booking/appointment CTA

### Blog (blog.html) - Optional
- Article cards with thumbnail, title, excerpt, date
- Categories / tags
- Pagination

## Section Component Patterns

### Navigation
- Logo left, links center or right
- Mobile hamburger menu with slide-in panel
- Sticky on scroll with backdrop blur
- Active page indicator

### Hero Variants
- **Split**: Text left, image/visual right
- **Centered**: Large headline centered, CTA below
- **Video background**: Overlay text on video/image
- **Slider**: Multiple slides with auto-advance

### Testimonials
- Carousel with photo, quote, name, company
- Grid of testimonial cards
- Single featured quote with large typography

### Footer
- Multi-column: about, links, services, contact
- Newsletter signup
- Social media icons
- Legal links (mentions legales, CGV, RGPD)
- Copyright

## Industry-Specific Additions

### Restaurant / Food
- Menu section with categories and prices
- Reservation widget/link
- Photo gallery of dishes
- Opening hours prominently displayed

### Medical / Health
- Services with detailed descriptions
- Doctor/practitioner profiles
- Online appointment booking
- Emergency contact info

### Real Estate
- Property listing grid with filters
- Property detail with gallery, specs, map
- Agent contact cards

### Artisan / Trade
- Before/after gallery
- Service area map
- Certifications and insurance info
- Quote request form

### E-commerce / Retail
- Product showcase
- Store locations
- Brand story
- Delivery/return info

### Tech / Startup / SaaS
- Feature comparison grid
- Integration logos
- Pricing table
- Demo/trial CTA
